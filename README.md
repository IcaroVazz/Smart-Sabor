Sistema de venda e gerenciamento de marmitas desenvolvido para a Secretaria de Educação, projetado para otimizar o controle de pedidos, administração de refeições e gerenciamento operacional de forma moderna, segura e eficiente.

A aplicação foi desenvolvida utilizando Next.js e Tailwind CSS, garantindo alta performance, renderização otimizada e uma experiência fluida de navegação em diferentes dispositivos. O sistema utiliza Firebase para autenticação de usuários, armazenamento e gerenciamento de dados em tempo real, proporcionando segurança, escalabilidade e integração eficiente entre clientes, administradores e equipes técnicas.

A plataforma centraliza processos operacionais relacionados à distribuição e comercialização de marmitas, oferecendo maior controle, praticidade e organização para toda a operação.

Principais características:

- Sistema de pedidos e gerenciamento de marmitas
- Interface moderna e responsiva
- Autenticação segura com Firebase
- Gerenciamento de dados em tempo real
- Arquitetura escalável e otimizada
- Navegação rápida com Next.js
- Painéis administrativos e operacionais

Tecnologias utilizadas:

- Next.js
- Tailwind CSS
- APIs REST
Firebase
JavaScript / TypeScript
